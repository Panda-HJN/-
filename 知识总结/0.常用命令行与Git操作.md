

#### 查看当前目录下文件
- 查看当前完整路径 PWD 
- 不包括隐藏文件 ls

- 查看当前目录下所有文件(包括隐藏文件) ls -a

- 查看当前目录下所有文件(包括隐藏文件)的详细信息 ls -al
#### 打开
- 打开根目录 cd /
- 打开家目录cd ~
- 打开上级目录 cd ..
#### 创建/删除/修改
- 创建文件 touch readme.md
- 删除文件 rm readme.md 
- 删除 node_modules 文件夹，不提示 rm -rf node_modules**慎用!!!!出过很多惨剧**
- 重命名 文件 mv readme.md README.md
- 创建文件夹 mkdir projects
